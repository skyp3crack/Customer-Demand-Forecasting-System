# Forecast Outputs

This directory holds the CSV files generated by the Python ML pipeline. The Node.js backend reads these files to import forecast data into MySQL.

## File Naming Convention

```
{forecast_type}_forecast_{model_type}_{YYYYMMDD}.csv
```

Examples:
- `daily_forecast_rf_20250101.csv`   — Daily forecasts for January 2025 (Random Forest)
- `monthly_forecast_rf_202501.csv`   — Monthly forecasts for 2025 (Random Forest)

## Generating Forecasts

```bash
# From project root
python main.py daily   --start-date 2025-01-01 --days 30
python main.py monthly --start-month 2025-01   --months 12
python main.py all     --model-type rf
```

## Importing into the Database (Admin)

After generating CSVs, import via the admin API endpoint:

```
POST /api/forecast/import-daily-batch?year=2025
Authorization: Bearer <admin-jwt-token>
```

Or import a single file:

```
POST /api/forecast/import-forecast
```

## CSV Schema — Daily Forecasts

| Column | Type | Description |
|---|---|---|
| `Drug` | string | Drug ATC code (e.g. `M01AB`) |
| `Date` | YYYY-MM-DD | Forecast date |
| `Predicted_Sales` | float | Predicted units |

## CSV Schema — Monthly Forecasts

| Column | Type | Description |
|---|---|---|
| `Year` | int | Year |
| `Month` | int | Month (1–12) |
| `M01AB` … `R06` | float | Predicted sales per drug |
